# davlo.io

Interactive space-themed website featuring an immersive 3D experience with planets, smooth animations, and dynamic theme transitions.

## Features

- Interactive 3D planet visualization with wheel-based zoom
- Dynamic theme system with seamless white ↔ black transitions
- Custom cursor with hover effects
- Animated star field with thousands of particles
- Smooth transitions powered by Framer Motion
- Desktop-optimized experience

## Tech Stack

Next.js • TypeScript • Tailwind CSS • Three.js • React Three Fiber • Framer Motion • WebGL

## Design Philosophy

Minimalistic yet immersive space aesthetic with a focus on smooth interactions and 3D graphics as the main experience.
